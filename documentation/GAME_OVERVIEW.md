# Game Overview

## Project Summary

Un-Dead Hotel is a web-based, real-time colony/survival management game set in an infinitely generated hotel environment ("Hilbert's").

The game mixes:

- world exploration,
- survivor management,
- scavenging and crafting,
- long-horizon base survival.

## Current Build Status

The codebase currently provides two runtime modes:

- `index.html`: procedural generation debug mode
- `game.html`: game runtime prototype mode

The procedural generation system is deterministic by chunk coordinate and world seed and is shared by both modes.

## Core Setting

- Infinite hotel layout generated by chunk.
- Multiple special spaces can appear repeatedly (gyms, restaurants, kitchens, spa, terraces).
- Outdoor-like terrace/balcony spaces are part of each generated floor.

## Gameplay Direction

### MVP Horizon

- Single-floor infinite world generation.
- Player-directed survivors.
- Basic zombie threat loop.
- Core scavenging/crafting for early survival.

### Post-MVP Horizon

- Vertical expansion (multi-floor traversal).
- Resource degradation over time (water/power instability).
- Late-game self-sustainability systems.
- Survivor behavior progression from direct control toward autonomy.
